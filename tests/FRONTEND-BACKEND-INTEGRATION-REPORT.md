## 🐛 问题发现与修复

### 问题描述
测试 6（点赞功能）在首次运行时失败，因为用户已经点赞过游戏。

### 修复方案
```javascript
// 添加状态检查逻辑
if (initialLiked) {
  // 如果已点赞，先取消点赞
  await fetch(`/api/v1/like/${gameId}`, {
    method: 'DELETE',
    headers: { 'Authorization': `Bearer ${token}` }
  });
}
// 然后再执行点赞操作
```

### 修复结果
- ✅ 测试现在可以智能处理已点赞状态
- ✅ 测试更加健壮
- ✅ 可以重复运行
