## 样式差异对照表

| demo.html (H5) | 小程序 (uni-app) | 说明 |
|---------------|-----------------|------|
| `:hover` 伪类 | `hover-class` 属性 | 小程序不支持 CSS :hover |
| `px` 单位 | `rpx` 单位 | rpx 是小程序响应式单位 |
| `position: fixed` | 支持但需谨慎 | iOS 小程序可能有兼容性问题 |
| `::-webkit-scrollbar` | 移除 | 小程序不支持自定义滚动条 |
| `-webkit-tap-highlight-color` | `hover-class` | 使用小程序触摸反馈机制 |
| `<div>` | `<view>` | 小程序使用 view 组件 |
| `<span>` | `<text>` | 小程序使用 text 组件 |
| `<button>` | `<button>` | 保留但样式需适配 |
| `onclick` | `@tap` | Vue 事件绑定 |
| `class` | `:class` | Vue 动态类名 |

## 主要改动点

### 1. hover 效果改造
```html
<!-- demo.html -->
<div class="game-card" onclick="playGame()">
  内容...
</div>
<style>
.game-card:hover { ... }
</style>

<!-- 小程序 -->
<view class="game-card" @tap="playGame" hover-class="game-card-hover">
  内容...
</view>
<style>
.game-card-hover { ... }
</style>
```

### 2. 单位转换
```css
/* demo.html */
padding: 20px;
font-size: 16px;

/* 小程序 (按 2x 转换，即 1px ≈ 2rpx) */
padding: 40rpx;
font-size: 32rpx;
```

### 3. 标签替换
```html
<!-- demo.html -->
<div class="container">...</div>
<span class="text">文字</span>

<!-- 小程序 -->
<view class="container">...</view>
<text class="text">文字</text>
```

### 4. 事件绑定
```html
<!-- demo.html -->
<button onclick="generateGame()">生成</button>

<!-- 小程序 -->
<button @tap="generateGame">生成</button>
```

## 注意事项

1. **小程序限制**
   - 不能使用 `eval()`
   - 不能使用 `window` 对象
   - 部分 CSS3 特性不支持（如滤镜）

2. **性能优化**
   - 避免频繁的 setData
   - 图片使用懒加载
   - 合理使用虚拟列表

3. **平台差异**
   - 不同小程序平台（微信/支付宝/百度）可能有差异
   - 建议使用 uni-app 条件编译处理平台差异
